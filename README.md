# Job_listing_website
Project Overview:
This project is a fully responsive and modern Job Listing Website that displays job openings as interactive cards. Users can filter jobs based on title, location, category, and experience level. The website updates results dynamically without page reload.
Features:
Job Cards:
- Displays job title, company, and location
- "View More" button opens a modal with detailed description
- Clean and modern card UI with hover effects
Filters & Search:
- Search by job title (keyword based)
- Filter by Location
- Filter by Category (IT, Design, Marketing)
- Filter by Experience Level (Fresher, Mid, Senior)
- Clear Filters button
- Dynamic filtering without page reload
Pagination:
- Displays limited jobs per page
- Previous and Next buttons
- Page number indicator
Help Desk Footer:
- Support email and phone
- Working hours
- Contact Support button
- Quick links section
Responsive Design:
- Mobile friendly layout
- Flexible grid system
- Adaptive filter section
Technologies Used:
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM Manipulation & Filtering Logic)
Project Structure:
index.html  → Main structure of website  
styles.css  → Styling and responsive design  
script.js   → Filtering, pagination & modal logic
